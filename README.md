# bug bounty notes

Personal notes, cheatsheets and references I use while hunting on [YesWeHack](https://yeswehack.com).

Not a tutorial, not a course — just stuff I write down so I don't have to google the same thing twice.

## contents

- [`web-security.md`](web-security.md) — XSS, SQLi, IDOR, SSRF, file upload, common headers
- [`oauth-notes.md`](oauth-notes.md) — OAuth 2.0 flows, JWT attacks, session/password reset
- [`linux-notes.md`](linux-notes.md) — privesc checklist, shells, file transfer, useful commands
- [`networking.md`](networking.md) — ports, nmap, DNS, SMB, tunneling/pivoting
- [`tools.md`](tools.md) — tools I use, flags I always forget, wordlists
